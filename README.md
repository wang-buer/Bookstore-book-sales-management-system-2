## 1、项目介绍

书店图书销售管理系统2实现了购物车、下单、订单详情、图书的增删改和订单管理以及登录注册，功能简单，适合技术学习


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可） 
- 硬件环境：Windows 或者 Mac OS
